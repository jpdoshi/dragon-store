# Dragon Store

Dragon Store is your gateway to transparent, libre and free open-source apps. It is exclusively made for power users to install mod apps easily.

The end goal of Dragon store is to provide apps to users in a safe and transparent way, directly from the source. I believe that users must know where they are getting the installer file and whether the source is legit or not. Dragon store use Repo-as-a-backend principle to let users download and install apps from latest release source.

## Features

Below shown are few features that are currently available in dragon store:

- Native app, not a web wrapper.

- Clean, Modern UI. No Ads. No Bloat.

- Optimized to run efficiently on mobile devices.

- Clear goal: transparent way to install FLOSS apps, directly from the source.

- Save your favorite apps with a simple click of a button.

- Respects user privacy. Doesn't connect to any external API.

- Apps are added almost daily to the library.

## Build Instructions

1. Clone the repo

   ```bash
   git clone https://github.com/jpdoshi/dragon-store.git
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start dev server

   ```bash
   npx expo start
   ```

4. Build production app

   Follow <a href="https://docs.expo.dev/guides/local-app-production/">Expo Build instructions</a> to build production app.
